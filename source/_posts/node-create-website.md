1、选择文件夹
2、执行:express  node（node是你指定的文件夹名，可随意指定），默认使用的是jade模块，如果想安装ejs模块，就使用这个命令：express -e  node
3、进入node （就是你刚才制定的文件夹名）目录：cd node
4、执行 npm install 安装node所需要的依赖模块。如果执行此命令出现错误，就使用npm install -g，前提是你现在是管理员的身份
5、至此，网站框架已搭建完成。
